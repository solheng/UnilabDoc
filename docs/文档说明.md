# 文档说明

## 文档简介

* 本文档的来源 - 基于aipod内部完备的功能函数接口.
* 本文档的目标 - 帮助各项目的开发同事快速搭建aipod流程并落地到项目使用中.
* 本文档的使用对象 - 具有一定Python基础语法能力的软件开发同事.
